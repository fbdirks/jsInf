[8] Keuzes maken: If

In computerprogramma’s moeten keuzes gemaakt kunnen worden door het programma. In een rekenmachine bijvoorbeeld mogen we een deling niet uitvoeren als de deler 0 is. Als het ochtend is mag de website niet ‘goedemiddag’ zeggen, maar moet het ‘goedemorgen’ zijn en als je inlogt met een fout wachtwoord mag er niet naar een volgende pagina gesprongen worden alsof je bent ingelogd.

Voor dit soort keuzes kent Javascript de commando’s

  • if/elseif/else
  • switch

Dit is een if/elseif/else voorbeeldje: 


if (time < 10) {
	greeting = "Good morning";
} else if (time < 20) {
	greeting = "Good day";
} else {
	greeting = "Good evening";
}
Maar algemeen kun je zeggen dat een if / else if / else blok er zo uit ziet: 

If (voorwaarde 1) {
		… één of meerdere code regels die worden uitgevoerd
		als de voorwaarde 1 waar is
	} else if (voorwaarde 2) {
		… één of meerdere code regels die worden uitgevoerd
		als de voorwaarde 2 waar is 
	} else {
		… code regels die worden uitgevoerd in alle andere gevallen.
	}

Een voorwaarde is de uitkomst van een vergelijking: 


	Leeftijd >= 16 ;  // is de leeftijd gelijk of hoger dan 16?
	Cijfer < 5 ; // Is het cijfer beneden de 5?
	Password == “geheim”; // Is het password gelijk aan geheim?
Een voorwaarde kan uit meer vergelijkingen bestaan, maar je moet die dan koppelen met een logische koppeling (&& = EN, || = Of): 

	(Naam == “Jan” && password == “geheim”)
	// is de naam Jan EN het password geheim?
Dit is wat anders dan: 

	(Naam == “Jan” || password == “geheim” ) 
	// is de naam Jan OF het password geheim?
In dit geval hoeft maar één van de voorwaarden (of beiden) waar te zijn om de hele voorwaarde waar te laten zijn.

Een ‘negatieve’ vergelijking kan ook: 


(Naam != “Jan”) // kijkt of een naam ongelijk aan Jan is

Oefeningen:

Hoe zien de voorwaarden er uit als we:
  1. Willen kijken of het vandaag “vrijdag” is?
  2. Willen kijken of een bedrag groter is of gelijk aan 100?
  3. Willen kijken of een afstand groter is dan 10km maar kleiner dan 20km?
  4. Willen kijken of een naam “Jan” is of “Jantje” of “Jantina”?
  5. Willen kijken of een klas NIET gelijk aan is “V4C”;

Opmerkingen:

Een if commando kun je gebruiken zonder een else if gedeelte, en zelfs zonder een else gedeelte.

Vergeet niet bij het bekijken of een waarde precies gelijk is aan iets anders een dubbel == te gebruiken. Bedrag==100 is wat anders dan Bedrag=100:

  • Bedrag == 100 is een vergelijking die waar of onwaar is
  • Bedrag = 100 is een toekenning en die is in JS altijd ‘waar’
Dus: 

Var wachtwoord=”geenidee”;
If (wachtwoord = “geheim” {
console.log(“Mag inloggen”);
} else {
console.log(“Mag niet inloggen”);
}

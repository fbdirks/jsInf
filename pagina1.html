Javascript Aantekeningen

1. Wat voegt Javascript toe aan HTML en CSS?

Javascript wordt tegenwoordig in toenemende mate ingezet:

  • (client-side) automatiseren van de browser
  • (client-side) beïnvloeden/veranderen van de vormgeving van een webpagina tijdens of na het laden, bv het zichtbaar of onzichtbaar maken van onderdelen van een pagina, of het sorteren van gegevens op een pagina, het aanpassen van de vormgeving aan het apparaat waarop een website bekeken wordt (responsible design), formuliervalidatie enz.
  • (server-side) alternatieve programmeertaal om servertoepassingen te maken (bv op basis van node.js).
  • (applicatie basis) automatiseren van applicatie (Photoshop, teksteditors enz.)

Javascript kan dus gezien worden als een mooie aanvulling op HTML en CSS. Het zorgt voor interactieve webpagina’s die kunnen reageren op invoer door de gebruiker. Omdat deze interactiviteit in de browser plaats vindt, dus bij de gebruiker zelf, hoeft er in de webserver geen rekening mee gehouden worden.

Omdat Javascript een populaire standaard geworden is wordt het in toenemende mate gebruikt als voertuig waarmee grote applicaties intern geautomatiseerd kunnen worden. Een goed voorbeeld is Google Sheets: je kunt GS aanvullen met eigen functies, maar die bouw je in Javascript.

Er zijn nogal veel voorgeprogrammeerde Javascript bibliotheken die bijzondere functionaliteit ontsluiten via Javascript: Node.js, Angular.js en jQuery om maar drie kanjers te noemen. Ook de Bootstrap heeft alles met Javascript te maken. Bootstrap speelt een belangrijke rol bij het bouwen van ‘responsive websites’, dus websites waarvan de vorm zich aanpast aan het apparaat waarmee de gebruiker de website benadert. Bijvoorbeeld DeStentor.nl.

[5b] Variabelen

In alle programmeertalen zijn variabelen van groot belang. Variabelen zijn stukjes geheugen waar je iets in kunt opslaan die je zelf een naam geeft. Door die naam te gebruiken kun je gegevens opslaan of ophalen uit het geheugen. Voorbeeld:


	Var leeftijd = 18;
	console.log(“De leeftijd is “ + leeftijd);

Als je een variabele voor de eerste keer gebruikt moet je de variabele ‘aanmelden’ bij javascript. Dit heet declareren. Je doet dit met het var commando:


	var test = 9;
	var naam;

In het tweede geval is de variabele wel aangemeld maar heb je er nog niets ingestopt. Dat komt dan later wel.

Met de ‘scope’ van een variabele bedoelen we het geldigheidsbereik. Als je een variabele meteen na de <script> tag declareert kan heel je JS code (ook de functies die je misschien schrijft) er gebruik van maken. Als je een variabele binnen een functie declareert en gebruikt is deze variabele alleen maar binnen die functie geldig.

Naamgeving van variabelen is in principe vrij. Kies altijd namen die duidelijk maken waarom of waarvoor je dat geheugen nodig hebt. Als je na een paar maanden weer naar oude programmacode kijkt zul je daar dankbaar voor zijn. Het is niet mogelijk om javascript opdrachten te gebruiken als naam van een variabele: 

Var for = 10 ;
is dus niet mogelijk!

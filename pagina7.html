[5c] Commentaar

Je kunt in JS code commentaar noteren die jou of iemand anders die de code leest misschien kan helpen de code beter te begrijpen. Commentaarregels worden door Javascript volledig genegeerd, ze worden niet uitgevoerd. Er zijn vooral twee soorten commentaar: Één regel commentaar: 

// dit is een 1-regelig commentaar
Meer regels commentaar: 

	/* 
		Dit is meerregelig commentaar.
		Zoveel regels als maar nodig zijn.
	*/
Een handige toepassing van commentaar is het ‘uitzetten’ van bepaalde code regels om te voorkomen dat ze worden uitgevoerd. In dit voorbeeld is de tweede regel op die manier even ‘uitgezet’: 

	Var test = 10;
	// console.log("Test = " + test);

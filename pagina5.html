[5] Wat zijn functions?

Functions zijn (meestal) kleine Javascript programma’s die een bepaalde actie uitvoeren op een webpagina of in een ander stuk javascript. In een Javascript gedeelte van een webpagina kunnen gerust meerdere functions gedefinieerd worden. Een function herken je aan deze vorm:


function mijnfunctie() {
	alert(“Dit is mijn functie”);
}

We zullen later zien wat die () in de naam van de functie betekenen en wat je er nog meer mee kan doen dan alleen maar neerzetten. Merk op dat de JS opdrachten die uitgevoerd moeten worden altijd tussen { } staan en dat iedere regel in zo’n blok afgesloten wordt door een ; .

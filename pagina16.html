Rekenen

JS kent dezelfde mathematische kernoperatoren als de meeste andere programmeertalen en rekenmachines. Dus:

  • uitkomst = 34 + 12; // som 46
  • uitkomst = 22 - 8; // verschil 14
  • uitkomst = 2 * 3; // vermenigvuldiging 6
  • uitkomst = 15 / 4; // deling, 3.75
  • uitkomst = 15 % 4 ; // restdeling, 3 dus

Onder een 'shorthand' operator verstaan we een bewerking die een verkorte vorm van opschrijven kent naast de 'normale' langere vorm. Als we bijvoorbeeld bij de variabele teller 1 willen optellen kunnen we dat zo doen: 

teller = teller + 1
De = betekent hier dus eigenlijk 'wordt gelijk aan'. Specifiek voor deze optelling (1 ergens bij optellen), die heel veel voorkomt, is een verkorte notatie heel gebruikelijk: 
teller++;
De notatie teller-- bestaat ook en is hetzelfde als: teller = teller - 1;

Belangrijke andere 'shorthand' operatoren van javascript zijn: 


prijs *= percentage ; // prijs = prijs * percentage;
lengte /= 2 ; // lengte = lengte / 2
bericht += " een toevoeging"; // bericht = bericht + " een toevoeging"
'Shorthand' operatoren hoef je niet te gebruiken. Dus als je je prettiger voelt bij de langere manier van opschrijven is dat prima.

Voor ingewikkelder mathematische functies, van het berekenen van de wortel tot en met het afronden van resultaten, kent JS de mathematische bibliotheek (deze zit standaard ingebouwd in javascript) die je bijvoorbeeld zo kunt gebruiken: 

uitkomst = Math.pow(8, 3); // 8 tot de macht 3
uitkomst = Math.sqrt(64); // de wortel van 64

[7] Hoe verander je een specifiek HTML element met Javascript?

Met Javascript kun je vrijwel ieder HTML element beinvloeden. Je kunt bijvoorbeeld de opmaak (kleur), de inhoud of de zichtbaarheid veranderen. Voor nu kijken we alleen even naar het veranderen van een heel specifiek HTML element. Je kunt een specifiek HTML element alleen maar veranderen als dat element een ‘id’ heeft:

<span id=”melding”></span>

De volgende functie plaatst de tekst “Pas op!” in deze <span> tag: 


	function uitkijken() {
		document.getElementById("melding").innerHTML = "<i>Pas op!</i>";
	}
Het groene gedeelte van de regel die met document begint is verplicht en betekent eigenlijk “Zoek op deze webpagina naar een element dat het volgende ID heeft”. Het deel tussen haakjes bevat dat ID en het deel na de = (innerHTML) geeft aan dat je de inhoud daarvan wilt veranderen Je mag hierin html tags gebruiken.

Merk op dat de tekst die achter de = zet in zijn geheel tussen " " moet staan.

Je kunt ook style elementen veranderen met een functie. Dat ziet er maar een klein beetje anders uit: 


	function rode_waarschuwing() {
		document.getElementById("melding").style.color = "red";
	}
Je kunt ook heel goed beide opdrachten in één functie zetten.

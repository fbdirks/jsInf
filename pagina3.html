Javascript Aantekeningen

[3] Waar plaats je een javascript programma(-onderdeel)?

Een javascript programma kan op de volgende plekken neergezet worden:

  • (traditioneel) in de <head> sectie van een webpagina tussen <script> tags. In de script tags hoeft geen verwijzing meer te staan naar javascript (wat voor html5 wel moest), javascript is de default.
  • (modern) in de <body>sectie van een webpagina, bij voorkeur aan het eind om het tonen van de HTML onderdelen te bespoedigen.
  • (professioneel) in een aparte file (met .js extensie) die in de <head> via het <script> commando wordt binnengelezen. Deze file kan ergens op het eigen filesysteem van de website staan, maar ook op een andere plek op het internet.



De eerste manier is vooral van belang bij functiebibliotheken van jezelf (filelink zoals hierboven), de tweede manier als je bibliotheken van anderen gebruikt.


Als je javascript gebruikt bij het automatiseren van een specifieke applicatie dan zal de plek waar je de scripts opslaat meestal door de applicatie voorgesteld worden. In de meeste gevallen wordt dan wel een extensie .js gebruikt, maar dat kan bij een specifieke applicatie ook anders geregeld zijn.

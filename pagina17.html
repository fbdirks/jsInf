17. Herhalingen

Herhalingsopdrachten laten je programma bepaalde instructies zolang herhalen als het moet. Een groep instructies die herhaalt wordt noemen we vaak een 'loop' (uitspreken als 'loep' - op zijn Engels). Bij alle herhalingen moet je altijd drie dingen weten:

  • Hoe hou ik bij hoevaak iets herhaald is (de teller)?
  • Wanneer moet de herhaling stoppen (de eindtest)?
  • Hoe verandert de teller iedere keer als er een herhaling gedaan is?
We spreken van een oneindige loop als de eindtest nooit gehaald zal worden.

Er zijn drie commando's in javascript die je opdrachten laten herhalen: FOR , WHILE en DO/WHILE.

Het for commando herhaalt een groep instructies een aantal keren en in de aanroep van het for commando herken je de drie hierbovengenoemde punten: de teller (met de startwaarde), de eindtest en de verandering van de teller: 


for (i = 0; i < 5; i++) {
	text += "The number is " + i + "
";
}
De teller is hier i, de teller start bij 0 en gaat net zolang door totdat de teller gelijk is aan 5. Iedere keer dat de instructies lopen wordt de teller 1 verhoogd (i++);

Bij het While commando wordt in de aanroep van While alleen de eindtest genoemd. Dat betekent dat je als programmeur er zelf voor moet zorgen dat de andere twee aandachtspunten (wat is de teller en hoe verandert de teller bij iedere loop?) voor elkaar zijn. Deze code voert hetzelfde uit als de for loop hierboven: 

 
var i = 0;
while (i < 5) {
	text += "The number is " + i + "
";
	i++;
}
Mocht je bijvoorbeeld de i++ in de loop vergeten dan kan je zomaar met een oneindige loop te maken krijgen. Soms is dat de bedoeling, bijvoorbeeld bij een alarm. 
 
var alarm = false;
while (!alarm) {
	alarm=controleer_deur();
}
Hierbij wordt verondersteld dat de functie controleer_deur de waarde true afgeeft als iemand de deur open doet. Als niemand de deur aanraakt blijft de While loop dus lang doorlopen..

Het commando DO / WHILE lijkt heel veel op WHILE maar het kenmerkende verschil is dat bij DO / WHILE de loop code altijd één keer wordt uitgevoerd voordat de eindtest wordt uitgevoerd. Soms heb je dat nodig voor het zetten van bepaalde variabelen. Deze code voert weer hetzelfde uit als de FOR en WHILE hierboven: 

 
var i = 0;
do {
	text += "The number is " + i + "
";
	i++;
} while (i<5);

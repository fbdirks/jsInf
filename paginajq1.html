[jQ1] jQuery 1

De javascript bibliotheek jQuery wordt overal op het internet gebruikt om het bouwen van interactieve webpagina's te versnellen en te vergemakkelijken. Met jQuery kun je bijvoorbeeld heel makkelijk de css van een element op een pagina laten veranderen (een input veld dat je vergeten bent in te vullen een opvallen rood kader geven bijvoorbeeld). Omdat jQuery zelf volledig uit javascript bestaat kun je betogen dat het gebruiken van jQuery eigenlijk overbodig is. Alles wat jQuery kan, kan javascript zelf natuurlijk ook. Het voordeel is alleen maar dat de ontwikkelaars van jQuery al die veel voorkomende acties al helemaal geprogrammeerd hebben en daarin ook nog eens rekening hebben gehouden met verschillen in browsers. Gebruik maken van jQuery maakt het dan voor anderen sneller en makkelijker om webpagina's te voorzien van dit soort effecten.

In deze paragraven worden alleen de basiselementen van het gebruik van jQuery behandeld.

Voorbereiden

Je kunt jQuery downloaden en ergens in het pad van de webpagina's zetten waar je het in wilt gebruiken. Daarbij kun je kiezen voor de gecomprimeerde (= onleesbare) versie en de ontwikkelaarsversie die wel leesbare javascript bevat. Normaal gesproken is de normale versie de beste. Downloaden kan op https://www.jquery.com

Een andere mogelijkheid is de jquery gebruiken die op andere sites wordt bijgehouden en in jouw scripts daar naar linken. Dit heet gebruiken maken van een Content Delivery Network (CDN). Op de site van jquery vind je de links die je dan moet gebruiken.

Het is hoe dan ook aan te raden thuis te raken op de site van jquery zelf, en met name ook het learning center op https://learn.jquery.com/ . Hier staat de documentatie en staat allerlei lesmateriaal.

Gebruik maken

Voordat je eigen scripts met jQuery elementen er in door een webpagina uitgevoerd kunnen worden moet de jQuery bibliotheek in je pagina worden ingelezen: 


	<script src="jquery.js"></script>
	<script> hier je eigen scripts </script>
Op de plek van jquery.js kan in dit voorbeeld ook de URL staan die verwijst naar de jquery in een CDN.

Skelet van een jQuery opdracht

jQuery stelt in de bibliotheek eigenlijk één object ter beschikking: jQuery. Dit object kan het makkelijkst aangesproken worden met het teken $. Dit is de alias van jQuery. In het skelet van een jquery opdracht is dit het eerste teken: 

$(selector).actie
Voor PHP veteranen is dit altijd even wat wennen. De andere twee dingen in het skelet zijn de selector, dat is de selectie van elementen op je webpagina waarop je een jquery actie wilt gaan uitvoeren en de actie, dus de handeling die je daarmee wilt uitvoeren. Je wilt bijvoorbeeld alle <p> onderdelen van een webpagina de tekstkleur blauw geven: 
$('p').css("color","blue");

Selectors & Filters

Selecteren in jQuery is een serieuze zaak. Er zijn verschillende, makkelijke maar ook complexe technieken om heel precies elementen op een webpagina te selecteren. Een paar voorbeelden:

  • per tag: $('p').
  • per tag met een bepaalde class: $('p.artikel').
  • per id: $("#paginanummer").
  • alle tags met een bepaald attribuut $( "input[name='voornaam']" );
  • meerdere tags: $('p , li').
  • bijzondere mogelijkheden, iedere oneven tabelrij: $( "tr:odd" ).
Een bepaald doel kan met al die selectiemogelijkheden soms op meerdere manieren bereikbaar zijn. Voor alle selectie en filtermogelijkheden wordt verder verwezen naar de documentatie van jQuery zelf.

Acties

Na het selecteren van een webpagina onderdeel moet je natuurlijk beslissen wat je er mee wilt doen. Mogelijkheden zijn bijvoorbeeld:

  • de css van een tag wijzigen: $('li').css("color","blue");
  • de html inhoud van een tag wijzigen: $('p#voetnoet').html("niet serieus nemen');
  • .. eventueel ook alleen de tekst: $('p#voetnoot').text("niet serieus nemen");
  • de value van een element wijzigen: $('input#achternaam').value("Vul iets in");
  • attributen toevoegen of wijzigen: $('p').attr("id","voetnoot");
  • elementen aan- of uitzetten, verwijderen, of animeren: $("li").hide();

Het is vaak de bedoeling dat javascript reageert op events die op de pagina gebeuren. Ook daar maakt jQuery het je makkelijker om events te benoemen. Het bekendste event is 


	$( document ).ready();
Dit is het event dat start als de de DOM van een pagina geladen is (het kan zijn dat plaatjes en dergelijke dan nog niet allemaal geladen zijn!). Binnen de ready komt meestal een anonieme functie (zie 17!). Ready is dus een methode van het jQuery object waarvan je zelf de code schrijft. Dit is een (klein) voorbeeldje. De anonieme functie begint na .ready(: 

$( document ).ready(function() {
    console.log( "ready!" );
});

Het aanmelden van meer events is een fluitje van een cent. Deze code komt over het algemeen in het $(document).ready() blok te staan (dat dus enorm kan uitgroeien!). Dit is een voorbeeld: 


$("p").click(function(){
 alert("Geklikt op een alinea (<p>)!");
});
Als iemand klikt op een alinea komt er een mededeling op het scherm.

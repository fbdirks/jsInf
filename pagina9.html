[6] Hoe start je een function?

Een function wordt gestart door een 'event', een gebeurtenis. Een heel goed te snappen event is het klikken op een knop. Een knop kan op een HTML pagina zo gedefinieerd worden: 

<button onclick="mijnfunctie()">Klik mij</button>
Het rode gedeelte is het gedeelte waarin wordt gezegd wat er moet gebeuren bij een klik op de knop, in dit geval is dat dus het starten van de JS function 'mijnfunctie'.

Een ander goed te snappen event is het onload event in de <body>. Onload is het event dat gestart wordt als een pagina helemaal geladen is. Dus: 

<body onload="groetjes()">
betekent dus dat de Javascript function ‘groetjes’ gestart moet worden als deze pagina helemaal geladen is.

Er zijn nog veel meer events, in totaal meer dan 50. Ze zijn niet allemaal belangrijk maar we gaan er wel een aantal kennen leren de komende tijd.

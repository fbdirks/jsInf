4. Wie voert de code uit?

De JS code wordt net als de HTML en CSS uitgevoerd door de browser zelf. Er is geen webserver voor nodig. Dit geldt natuurlijk niet voor code die hoort bij servertoepassingen (bijvoorbeeld in een node.js applicatie). Omdat de code in de browser wordt uitgevoerd kun je een JS pagina dus ook starten vanuit de Windows verkenner door er op te dubbelklikken. In het geval van automatiseren van applicaties met JS voert natuurlijk deze applicatie in combinatie met de standaardbibliotheek van JS deze scripts uit. De verantwoordelijkheid hiervoor ligt natuurlijk geheel bij de bouwer van de applicatie.

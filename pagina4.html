4. Wie voert de code uit?

De JS code wordt net als de HTML en CSS uitgevoerd door de browser zelf. Browsers maken hiervoor gebruik van zogenaamde javascript 'engines'. Er is geen webserver nodig.

Dit geldt natuurlijk niet voor code die hoort bij servertoepassingen (bijvoorbeeld in een node.js applicatie). In feite bouw je dan met node.js een webserver op waar anderen contact mee kunnen gaan leggen. Er is niet extra PHP, Python of Apache voor nodig.

Omdat normaal gesproken de javascriptcode gewoon in de browser wordt uitgevoerd kun je een JS pagina dus ook starten vanuit de Windows verkenner via een dubbelklik.

In het geval van automatiseren van applicaties met JS, bijvoorbeeld een editor of PhotoShop, voert natuurlijk deze applicatie in combinatie met de standaardbibliotheek van JS deze scripts uit. De verantwoordelijkheid hiervoor ligt geheel bij de bouwer van deze applicaties.

[5d] Andere basissyntax

De basissyntax van javascript is niet moeilijk.

Iedere JS regel wordt afgesloten met een ; om aan te geven dat het commando klaar is. 


var x = "deze regel is niet af"
var y = "deze regel is wel af";

Aanhalingstekens " en ' mag je beiden gebruiken en hebben beiden hetzelfde effect. Je mag ze niet mixen. Dit mag dus wel: test = "hallo" of test = 'Hallo', maar dit niet: test = "Hallo'.


var groet = "hans";
groet = 'hans';
groet = "hans'; // dit is niet correct (let op de syntaxkleuren)

JS is hoofdlettergevoelig. Een variabele test is dus niet hetzelfde als de variabele Test. 


var test = "OK";
var Test = "Niet OK";

Een blok javascript code regels die bij elkaar horen kun je tussen accolades zetten: { } . Het aantal regels dat tussen accolades mag staan is in principe onbeperkt. 


{
	console.log("blok 1");
	x++;
}

console.log() print iets uit naar de javascript console. Die kun je in een browser meestal openen door F12 in te drukken. (Op school werkt dat niet altijd..)

Op deze manier kun je de inhoud van een variabele printen: 

console.log("in variabele x zit: " + x);
